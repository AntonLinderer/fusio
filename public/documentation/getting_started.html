Authentication

Fusio uses OAuth2 for authorization. In order to obtain an access token you have to send an client credentials request to the following endpoint. The app key and secret must be obtained by the API provider.

POST /authorization/token

To check whether the authentication was successful you can make an GET request to the endpoint.

GET /authorization/whoami

If the access token is no longer needed you can revoke the access by making an POST request to the endpoint.

POST /authorization/revoke
